Lab 5: Harris Corner Detection

Sun 16 February 2023 by Aditya Arora

The goal of todays lab is to implement the Harris corner detector. Please see the 'Image_features' slide for a walkthrough of the algorithm. Given an image, the Harris corner detector returns an image of the same size where there is a 1 in all pixel locations with a corner and a 0 otherwise.

Your todo Your job is to make two functions: 1) harris_response() and 2) local_max_and_thresh(). These functions compute the corner response scores and perform the thresholding, respectively. Once you are finished, try your algorithm out on a few images to see if it works as expected. Use k=0.04

The harris_response should 

Compute the Harris cornerness function.
Function does not do any normalization or nms. 

Args:
    input: input image with shape :math:`(B, C, H, W)`.
    k: the Harris detector free parameter.
    grads_mode: can be ``'sobel'`` for standalone use or ``'diff'`` for use on Gaussian pyramid.
    sigmas: coefficients to be multiplied by multichannel response. Should be shape of :math:`(B)`
    It is necessary for performing non-maxima-suppression across different scale pyramid levels.

Return:
    the response map per channel with shape :math:`(B, C, H, W)`.

1) take gradients of input using 'spatial_gradient()'
2) blur dx and dy using 'gaussian_blur2d()'
3) calculate the determinant and trace using dx2, dy2, and dxy
4) compute the response map (i.e., difference between the determinant and k*trace^2)

The local_max_and_thresh should 

find the local maxima and threshold, then return the binary map of 0's and 1's where all 1's are corners
