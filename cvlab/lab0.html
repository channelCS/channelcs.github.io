Lab 0: Setting-up your workspace

Sun 15 January 2023 by Aditya Arora

In this lab, we would learn setting up our workspace and understanding basic image operations. Follow these steps to set up your own laptop. The instructions are also same for all Operating Systems, otherwise the differences are provided.

  1. We are going to use Anaconda for setting up our Python workspace. Go to Anaconda Website and download and install the latest version of Anaconda for your Operating System. Follow the instructions online to install Anaconda. Once done, for Windows, open Anaconda Prompt and for MacOS and Linux, open Terminal. If you installation was correct, you would see 

    2. (base) $ // This is the base environment

  2. We are now going to create a virtual environment so that all our installations do not affect the base environment. For creating a new virtual environment, run 

    3. (base) $ conda create -n cvlab_env python=3.8 -y

  3. Once the virtual environment is created, activate it using 

    4. (base) $ conda activate cvlab_env

  4. For the course of the labs, we would require certain python packages. We would install them by running 

    5. (cvlab_env) $ pip install torch==1.12.0 torchvision==0.13.0 numpy==1.24.1 matplotlib==3.6.3 jupyter==1.0.0 opencv-python==4.7.0.68

  5. Make a new directory for this lab named cvlab_dir using 

    6. (cvlab_env) $ mkdir cvlab_dir
    (cvlab_env) $ cd cvlab_dir
    (cvlab_env) $ mkdir lab0
    (cvlab_env) $ cd lab0

  6. We are going to check if our installations were done correctly. For this we would run a jupyter notebook, load and show an image. Launch a notebook using 

    7. (cvlab_env) $ jupyter notebook

  7. You would see a browser window open. Click on New > Python 3. Also download an image in lab0 folder.
  8. In the Cell, write 

    9. import matplotlib.pyplot as plt
    import numpy as np

  9. Load image in img variable and show it. 

    10. img = plt.imread('image1.jpg')
    plt.imshow(img)
