Lab 1: Python Warm-up

Sun 15 January 2023 by Aditya Arora

This lab is designed to make sure you can load an image, manipulate the values, and produce some output using Python. Plenty of additional Python tutorials can be found by searching the web. This lab is to be done individually. Try to avoid using loops which dealing with large arrays (e.g., images) as much as possible. Instead try to use vectorized operations by applying a single Python command to an entire array. To get documentation for a particular Python function, visit the Python documentation. Finally, make sure you do the appropriate typecasting (i.e., uint8 and float32) when working with images.

The two libraries you will work with today are Numpy and Matplotlib. Numpy is a library designed to handle arrays of numbers with many dimensions. Matplotlib is an image processing and visualization library which works with numpy arrays. See this tutorial for a quick introduction on how to use both together and perform some basic operations.

If you have any questions or if you are finished with the lab, please let the TA (Aditya Arora) know.

If things break (or if you get disconnected from the kernel) and you need to restart your kernel, click `kernel - restart' and this will delete all variables and `unrun' any cells.
